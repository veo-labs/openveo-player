OpenVeo Player requires additional elements for development :

  • Git - openveo-player is versioned with git
  • Ruby / Sass / Compass - CSS is written using SASS / Compass
  • Karma - Unit tested are performed using karma
  • Grunt - Grunt is used to launch unit tests and compile sources
